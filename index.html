How I improved the website

To improve the cross-course project I focused on completing the user journey by adding another checkout-page and a completed checkput-page. I changed how the checkout-form works in terms of form-errors to be able to take the user to the next page only when the user provided the needed information. If found it hard to add the error messages as the user fills out the form, so I added placeholders to show the user how to fill in the different fields to prevent the user from being confused. The code of this isnt very realistic but that was the only way I found it to work partly as I wanted.

I also removed the select form on the jacket-specific-page and replaced it with radio buttons so that one of the sizes were checked by default, which made more sense.

I changed some of the button's values and added a transition to hover on click-able items. I also added loaders and changed the main element if there are any errors in JS. I also added a footer to make it more realistic.

My thoughts on the year and my progress

I didn't know what to expect when I started this year considering that I've never actually had any experience with coding or computers in general. As a complete beginner it was hard to understand how everything worked, there were a lot of "basic" terms that were completely new to me so it really was a process to grasp the whole subject.

I've defintaly learned that the more you practice yourself - the more you learn. I found it to be hard to read about coding and to try and understand it in my head. Testing it out and seeing what worked and what didn't made it easier to understand. I've really started to understand CSS a bit more than I used to. JavaScript is the language I struggle with, and I find it to be somehow similar to math which I also struggle with. I am starting to notice some improvement though, so I just have to practice more.
